This package is the starting point for your application to use the jvCard framework. Create a new instance of vCard to start from scratch.

vCard vc = new vCard();
vc.addType(new VERSION30());
vc.addType(new N("Brandt", "Thilo"));
System.out.println(vc.toString());
Starting with an InputStream you should use the static method

InputStream is = ...; // get a vCard as InputStream e.g. from a file
List myvcards = vCard.parsevCard(is);
for (vCard vc : myvcards)
System.out.println(vc.toString());
