This package supports creating an manipulating of vCard elements' parameters. It follows the IETF specification as defined in http://www.ietf.org/rfc/rfc2426.txt

IType e = ...; // get element from vCard object
IParameter p = e.getParameter(IParameter.PARAMETER_NAME_TYPE);
if (p!=null) {
System.out.println(p.getValue());
}
